# 🧠 Campus Surveillance Brain

An AI-powered intelligence system that detects suspicious activity patterns, analyzes threat levels in real-time, and visualizes behavioral insights through an interactive dashboard.

---

## 🚀 Features

- 🔍 AI-based Threat Detection (NLP)
- 📊 Activity Timeline Visualization
- 🚨 Real-Time Alert System (Spike Detection + Sound Alerts)
- 👤 Suspect Profiling Engine (Risk Classification)
- 🧠 Pattern Analysis (Keyword Frequency + Behavior Trends)
- 📄 Exportable Intelligence Reports
- 🕶️ Hacker-style Dashboard UI

---

## 🧠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js + Express  
- **AI/NLP:** HuggingFace Inference API  
- **Data Handling:** In-memory (extendable to Firebase/DB)  

---

## ⚡ How It Works

1. User inputs text (simulated social media data)
2. AI model evaluates threat probability
3. Data is stored and analyzed for patterns
4. System detects abnormal spikes in activity
5. Dashboard visualizes insights + triggers alerts

---

## 🔥 Use Case

This project simulates a **mini OSINT (Open Source Intelligence) system** that can be extended for:

- Campus security monitoring  
- Social media threat detection  
- Fraud / abuse pattern detection  
- Intelligence dashboards
